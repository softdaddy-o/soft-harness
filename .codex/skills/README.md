# Project Skills

Project-local Codex skills for `F:\src3\Docs`.

## Daily Workflow

- `daily-start`: orchestrate the morning scrape-to-post workflow
- `scrape`: run daily social scraping, summaries, and SR updates
- `find-sources`: discover cited sources and maintain scrape candidates
- `analyze-ideas`: mine product ideas from `devideas` scrape output
- `zettelkasten-process`: ingest journals or Todoist links into Logseq notes
- `learn-from-posts`: compare scheduled Threads posts with actual output
- `schedule-threads`: select and schedule new Threads posts
- `aiki-publish`: generate and publish AIKI news articles
- `english-drill`: generate daily English drill sentences from feedback history

## Notes

- These skills are the Codex-local migration of the repo's legacy Claude workflow.
- Skill bodies were restored from the original UTF-8 Claude sources and normalized for Codex metadata.
- Legacy slash-command references in evals and guidance were rewritten to the matching local skill names.
